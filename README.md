<h2>ECHO66</h2>
<hr><br>
<p>
A (yet) very simple, x86 universal operating system. 
Runs exclusively in 16-bit Real Mode, and was written in FASM flavoured assembly.
So far we have a relatively finished bootloader (originally written on paper during history class), and a kernel thats capable of echoing back keyboard interrupts. *Énorme*</p>
